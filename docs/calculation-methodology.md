# 計算方法論

このドキュメントでは、Ecodan省エネ計算シミュレーターで使用されている主要な計算方法論について説明します。

## 1. 物理定数

シミュレーションの基礎となる物理定数は以下の通りです。

| 定数 | 値 | 説明 | 参照元 |
| :--- | :--- | :--- | :--- |
| `BTU_PER_THERM` | `100_000` | 1サーモあたりのBTU（英国熱量単位） | 業界標準 |
| `BTU_PER_KWH` | `3_412` | 1 kWhあたりのBTU | 業界標準 |
| `CO2_LB_PER_THERM` | `11.7` | 1サーモあたりのCO2排出量（ポンド） | EPA |
| `CO2_LB_PER_KWH` | `0.386` | 1 kWhあたりのCO2排出量（ポンド） | eGRID 2023 |
| `CO2_LB_PER_TON` | `2000` | 1トンあたりのポンド数 | 業界標準 |

## 2. COP (Coefficient of Performance) 補間

ヒートポンプの成績係数（COP）は外気温によって変動します。シミュレーターでは、以下のCOPルックアップテーブルを線形補間してCOPを推定します。

**COPテーブル:**

| 外気温 (華氏) | COP |
| :--- | :--- |
| -5 | 1.5 |
| 5 | 1.8 |
| 17 | 2.5 |
| 30 | 3.1 |
| 40 | 3.6 |
| 47 | 4.0 |
| 55 | 4.5 |
| 65 | 5.0 |

このデータはASHRAE（米国暖房冷凍空調学会）の寒冷地ヒートポンプ性能データに基づいています。

## 3. AFUE (Annual Fuel Utilization Efficiency) 推定

既存のガス炉の年間燃料利用効率（AFUE）は、その年数によって推定されます。以下のロジックが適用されます。

| 年数 | AFUE |
| :--- | :--- |
| <= 5年 | 0.96 |
| <= 10年 | 0.90 |
| <= 15年 | 0.85 |
| <= 20年 | 0.80 |
| <= 25年 | 0.78 |
| > 25年 | 0.75 |

この推定はENERGY STARの年齢別テーブルに基づいています。

## 4. 設計温度 (Design Temperature) 推定

入力された郵便番号の上3桁に基づいて、その地域の設計温度（華氏）を推定します。これは、米国の郵便番号と設計温度の経験的なマッピングに基づいています。

## 5. 州ごとのエネルギー価格

EIA（米国エネルギー情報局）の2024年のデータに基づいた、各州の住宅用ガス価格（ドル/サーモ）と電気料金（ドル/kWh）を使用します。郵便番号から州を特定し、その州のエネルギー価格を計算に利用します。該当する州が見つからない場合は、米国の平均価格が使用されます。

## 6. 主要な計算ステップ

`runCalc`関数は、以下の主要なステップで計算を実行します。

1.  **AFUE, 設計温度, COP, 州のエネルギー価格の取得**: 入力値に基づいて、それぞれの値を推定・取得します。
2.  **月々のガス使用量 (サーモ)**: `月々のガス料金 / ガス価格`
3.  **月々のガス暖房によるBTU**: `月々のガス使用量 * BTU_PER_THERM * AFUE`
4.  **月々のヒートポンプ電力消費量 (kWh)**: `月々のガス暖房によるBTU / (BTU_PER_KWH * COP)`
5.  **月々のガス暖房コスト**: 入力された月々のガス料金
6.  **月々のヒートポンプ暖房コスト**: `月々のヒートポンプ電力消費量 * 電気料金`
7.  **月々の節約額**: `月々のガス暖房コスト - 月々のヒートポンプ暖房コスト`
8.  **年間節約額**: `月々の節約額 * 12`
9.  **CO2排出量削減**: ガス暖房とヒートポンプ暖房それぞれの年間CO2排出量を計算し、その差を算出します。
    *   年間ガスCO2排出量: `月々のガス使用量 * 12 * CO2_LB_PER_THERM / CO2_LB_PER_TON`
    *   年間ヒートポンプCO2排出量: `月々のヒートポンプ電力消費量 * 12 * CO2_LB_PER_KWH / CO2_LB_PER_TON`
    *   CO2削減量: `年間ガスCO2排出量 - 年間ヒートポンプCO2排出量`

## 7. 税額控除と投資回収期間

-   **税額控除**: 導入コストの30%を税額控除として計算しますが、上限は$2,000です（IRA §25Cに基づく）。
-   **純粋な導入コスト**: `導入コスト - 税額控除`
-   **投資回収期間**: `純粋な導入コスト / 年間節約額`。年間節約額が正の場合に計算されます。

## 8. 10年間の累積節約額

将来のエネルギー価格上昇を考慮し、年間節約額が毎年2%ずつ増加すると仮定して、10年間の累積節約額を計算します。
