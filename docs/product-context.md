# Ecodan 製品コンテキスト

このドキュメントでは、シミュレーターの対象となる三菱電機 Ecodan ヒートポンプ給湯機に関する背景情報を提供します。

## 1. Ecodan とは

Ecodan は、三菱電機が提供する空気熱源ヒートポンプ給湯暖房システムです。外気の熱を利用して水を温めることで、従来のガスボイラーや電気ヒーターと比較して高いエネルギー効率を実現し、運用コストとCO2排出量の削減に貢献します。

## 2. 主要な特徴

-   **高効率**: 外気の熱を利用するため、投入電力以上の熱エネルギーを供給できます。これにより、エネルギー消費量を大幅に削減します。
-   **環境性能**: 化石燃料の燃焼を伴わないため、CO2排出量を削減し、地球温暖化対策に貢献します。
-   **多様な設置オプション**: 屋外ユニットと屋内ユニットを組み合わせることで、様々な住宅環境に対応可能です。
-   **暖房と給湯**: 暖房だけでなく、家庭用給湯も賄うことができます。

## 3. シミュレーターの目的

このシミュレーターは、Ecodan システムを導入した場合に、ユーザーが期待できる経済的メリット（月々の節約額、年間節約額、投資回収期間、10年間の累積節約額）と環境的メリット（CO2排出量削減）を定量的に示すことを目的としています。

## 4. シミュレーションの前提条件

シミュレーションは、以下の一般的な前提条件に基づいて行われます。

-   **既存システム**: 主にガスボイラーからの切り替えを想定しています。
-   **地域特性**: 郵便番号に基づいて、地域の気候条件（設計温度）とエネルギー価格（ガス、電気）を考慮します。
-   **税制優遇**: 米国のIRA §25Cなどの税額控除を考慮に入れます。
-   **エネルギー価格の変動**: 10年間の累積節約額の計算では、エネルギー価格が毎年一定の割合で上昇するという仮定を置いています。

## 5. 免責事項

このシミュレーターの結果はあくまで推定値であり、実際の節約額やCO2排出量削減量は、個々の住宅の断熱性能、居住者の生活習慣、気象条件、エネルギー価格の変動など、様々な要因によって異なります。最終的な導入判断は、専門家との相談の上で行うことを推奨します。
