# Ecodan Energy Saving Calculator

Interactive web tool to estimate potential energy savings when switching from conventional heating systems to Mitsubishi Electric Ecodan Air-to-Water Heat Pump systems.

This tool is designed primarily for:

- HVAC contractors
- Distributors
- Sales engineers
- Homeowners exploring heat pump adoption

The calculator provides an intuitive way to demonstrate how heat pump systems can reduce energy consumption and operating cost under typical North American conditions.

---

## Purpose

The goal of this project is to support **Ecodan product communication and education** by visually demonstrating potential energy savings compared to conventional heating systems.

This tool is not intended to replace full HVAC system design or load calculation software.

Instead, it acts as a **conversation starter and conceptual estimation tool**.

Typical use cases:

- Contractor explaining heat pump benefits to homeowners
- Distributor training sessions
- Sales engineers demonstrating operating cost differences
- Educational use during product presentations

---

## Key Features

- Simple heating system comparison
- Estimated annual energy consumption
- Estimated operating cost comparison
- Visual charts for intuitive understanding
- Modern UI optimized for quick explanation during sales conversations

---

## Assumptions

This calculator uses simplified assumptions including:

- Average heating load estimates
- Typical system efficiency values
- Simplified COP estimates for heat pump operation
- Average electricity and fuel prices

Actual savings depend on many factors including:

- Climate zone
- Building insulation level
- Heating system design
- Energy prices
- System sizing
- Control strategy

Results should therefore be interpreted as **illustrative estimates only**.

---

## Limitations

This tool does NOT perform:

- Manual J load calculations
- Detailed hydronic system design
- Dynamic weather simulation
- Equipment sizing

For detailed design, professional HVAC calculation tools should be used.
